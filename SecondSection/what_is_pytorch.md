<h1>什么是 PyTorch?</h1>
PyTorch 是一个基于 Python 的科学计算包，主要定位两类人群：
<ul>
 	<li>NumPy 的替代品，可以利用 GPU 的性能进行计算。</li>
 	<li>深度学习研究平台拥有足够的灵活性和速度</li>
</ul>
